# 2. Lea sobre el resto de métodos de la clase File de Python aquí y cree una tabla donde explique qué hace cada uno. No necesita usar código para esto, es solo crear una tabla en Notion o Word.

read	Lee todo el contenido del archivo como un solo texto
readable	devuelve un True si el archivo se puede leer
readline	Lee y devuelve una solo linea del archivo
readlines	lee todas las lineas del archivo y las devuelve como una lista
seek	Mueve el cursor a otro lugar del archivo.
seekable	Devuelve un True si el archivo permite mver el cursor. Seek
tell	devuelve donde está el cursor actual en el archivo con caracteres
truncate	corta el archivo desde cierta posición. Puede usarse para borrar parte del contenido
writable	Devuelve True si el archivo se puede escribir
write	Escribe un text en el archivo. Si no se agrega un salto de linea manual, todo seguido
writelines	escribe una lista de lineas en el archivo. No agrega saltos de línea automaticos.
